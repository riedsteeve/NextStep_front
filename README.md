# NextStep

NextStep est une application web de gestion de candidatures conçue pour simplifier et centraliser le suivi de vos recherches d'emploi.

L'application permet d'ajouter, modifier et supprimer des candidatures, de suivre l'évolution de leur statut (en cours, entretien, refusée, acceptée) et d'organiser efficacement votre parcours professionnel.

---

## Stack Technique

Le projet est développé avec les technologies suivantes :

- **Front-end :** [Vue.js 3](https://vuejs.org/) (Vite, TypeScript, Tailwind CSS, Vuetify)
- **Back-end :** [Express.js](https://expressjs.com/) (Node.js)
- **Base de données & Authentification :** [Supabase](https://supabase.com/)

---

## Fonctionnalités Principales

- **Gestion des candidatures :** CRUD complet (Création, Lecture, Mise à jour, Suppression).
- **Suivi des statuts :** Visualisation rapide de l'état de chaque démarche.
- **Tableau de bord :** Vue d'ensemble sur les recherches en cours.
- **Gestion des mémos :** Ajout de notes pour préparer les entretiens ou relancer les recruteurs.

---

## Installation et Lancement du Front-end

### 1. Prérequis

Assurez-vous d'avoir [Node.js](https://nodejs.org/) installé sur votre machine (version 20+ recommandée).

### 2. Cloner le projet

```bash
git clone [https://github.com/riedsteeve/NextStep_front.git](https://github.com/riedsteeve/NextStep_front.git)
cd NextStep_front
```
