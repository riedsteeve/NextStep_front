# NextStep_front
NextStep est une application web de gestion de candidatures permettant de suivre facilement l’évolution de ses recherches d’emploi.

Elle permet d’ajouter, modifier et supprimer des candidatures, de suivre leur statut (envoyée, entretien, acceptée, refusée) et d’organiser efficacement son processus de recrutement.

Le projet est composé d’un frontend en Vue.js et d’une API en Express.js, avec une base de données gérée via Supabase.
